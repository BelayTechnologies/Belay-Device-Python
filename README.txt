This is a basic readme file that will outline what is needed to get things rolling.

ESXi Server

vCenter Server

You will need Python 2.7+ (recommend ActivePython)
Python requirements:
	pyvmomi

Folders included here:
	pyDevice - Base directory for containing all Python code for Belay Device
	pyDevice\agent - Code pertaining directly to the agent(s)
	pyDevice\database - Code pertaining directly to the database
	pyDevice\manager -  Code pertaining directly to the manager
	sut-scripts - Scripts that will be run on the system(s) under test
	regression-tests - Code that has been tested and verified working
	user-interface - Code pertaining directly to the UI
